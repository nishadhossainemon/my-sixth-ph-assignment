Project Name : Digital Product Marketplace (DigiTools)

Description : 
A modern digital product marketplace where users can browse, add to cart, and purchase premium AI tools, templates & productivity software — all in one place.

Tech Used :
. React 19 · Vite  Tailwind CSS · DaisyUI · React Toastify · React Icons


3 Key Features :

Smart Cart — Add/remove products with live cart count & toast notifications

Suspense Data Fetching — Async product loading using React Suspense & use() hook

Clean UI — Fully responsive SaaS-style design with Pricing, Steps & CTA sections
